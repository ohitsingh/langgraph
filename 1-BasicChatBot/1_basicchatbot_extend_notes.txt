Important improvement
---------------------

Your current graph has this edge:

builder.add_edge("tools", END)

That means the graph stops immediately after a tool runs.

For a more natural chatbot response, you can change it to:

builder.add_edge("tools", "tool_calling_llm")

Why this is useful:
After the tool runs, the result goes back to the LLM. Then the LLM can read
the tool result and produce a final human-friendly answer.

Improved flow:

START -> tool_calling_llm -> tools -> tool_calling_llm -> END
